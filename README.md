радіотехнічної системи

Обладнання: STM32H743VIT6, CubeMX, STM32CubeIDE.

Мета: Ознайомитися з принципами роботи аналого-цифрового перетворювача STM32F411,
його характеристиками та реалізувати програму для зчитування аналогових сигналів у режимі
реального часу.

Завдання до лабораторної роботи:

Варіант 5. Зчитати сигнал з каналу АЦП (PA5), перетворити його у напругу та відображати 
на світлодіоді PD13: якщо напруга > 1.7 В — світить, інакше вимкнений.

Загальний опис функціоналу 💡

Програма реалізує систему безперервного моніторингу аналогового сигналу у режимі реального часу. 
Логіка роботи алгоритму:
- Таймінг та вибірка: Замість блокуючої затримки використовується системний таймер HAL_GetTick(), який кожні 100 мс ініціює одиночне перетворення АЦП (ADC1).
- Перетворення: Зчитаний цифровий код із регістра даних перетворюється на реальне значення напруги за формулою: $Voltage = \frac{ADC\_Value \times V_{ref}}{Resolution}$. Для 12-бітного режиму та опорної напруги 3.3 В це виглядає як $Voltage = \frac{ADC\_Value \times 3.3}{4096}$.
- Пороговий компаратор: Отримана напруга (float voltage) порівнюється із заданим порогом у 1.7 В.
- Індикація: Якщо напруга перевищує поріг, подається логічна «1» на пін PD13 для увімкнення світлодіода. Якщо падає нижче — світлодіод вимикається.

Схема підключення ⚙️
- Підключення потенціометра: Лівий вивід: підключити до GND (земля мікроконтролера).
2. Правий вивід: підключити до 3.3V (живлення мікроконтролера).
3. Центральний вивід (повзунок): підключити до аналогового входу PA5 (ADC1_IN). При обертанні ручки на цьому піні буде плавно змінюватися напруга від 0 до 3.3 В.2.
- Підключення індикації: LED2 (PD13): GND → Катод LED → Анод LED →  резистор (220 Ом) → пін PD13.
